Bienvenido al simulador

Este proyecto contiene dos implementaciones previstas para el parcial:

  • Implementación 1: Asignación de memoria con estrategias First Fit, Best Fit y Worst Fit.
  • Implementación 2: Traducción de direcciones virtuales a físicas (pendiente).

El programa principal se ejecuta desde la consola. Después de cada acción, se actualiza el reporte HTML que puede consultar aquí:

  • MemoryAllocation/memory_report.html

Cuando exista la segunda implementación, el reporte correspondiente también estará disponible desde esta página.

Inicie el programa C++ desde la terminal, elija el modo interactivo o por archivo, y luego recargue el reporte HTML para ver los resultados.
